css选择器

发表于 2017-02-13   |   分类于 css3   |  

简单的介绍十一种css选择器

一、id选择器

1.同一页面的id名称不可重复
2.id由#开头，不能以数字开头，名称不能与标签选择器相同
3.如果是为了设置样式，一般不会使用id，id是留给js使用的

二、class选择器

1.同一页面的class名称可重复
2.在html中，每个标签可以同时绑定多个类名class=”c1 c2”

三、后代选择器

格式：标签1 标签2{属性：值;}
1.先找到标签1，再找标签2，再设置属性
2.有多种组合：标签-标签，id-标签，id-class，class-标签，id-id，class-class
3.后代选择器可以无限延伸，不止两层

四、子元素选择器

1.作用：找到指定标签中所有特定的直接子元素，然后设置属性
2.格式：标签1>标签2{属性：值;}
3.子元素选择器只会查找到指定标签，不会影响到很里面的标签
4.有多种组合：标签-标签，id-标签，id-class，class-标签，id-id，class-class
5.子选择器可以通过>一直延续下去，如：div>ul>li>p{}

五、交集选择器（没有连接符号）

1.作用：给所有选择器选中的标签中，相交的那部分标签设置属性
2.格式：选择器1选择器2{属性：值；}
3.选择器之间没有连接符号
4.id，标签，class可任意组合
5.交集选择器仅作为了解，实际开发中运用不多

六、并集选择器

1.给所有选择器选中的标签设置属性
2.格式：选择器1，选择器2{属性：值；}
3.id，标签，class可任意组合

七、兄弟选择器

1.只能选中同级关系的标签
2.相邻兄弟选择器：给指定选择器后面紧跟的那个选择器选中的标签设置属性。格式：选择器1+选择器2{属性：值；}
3.通用兄弟选择器：给指定选择器后面的所有选择器选中的所有标签设置属性。格式：选择器1~选择器2{属性：值；}

八、序选择器

格式1：p:first-child{color: red;}
作用1：选中同级别中的第n个标签，不区分类型

first-child 选中同级别中的第一个标签
last-child 选中同级别中的最后一个标签
nth-child(n)选中同级别中的第n个标签
nth-last-child(n)选中同级别中的倒数第n个标签
only-child 选中父元素中唯一的子元素（例：只有一个p，不存在其他元素才可选中）
only-of-type 选中父元素中唯一的制定元素（只存在一个p，可存在其他标签，就可选中）
nth-child(odd)选中同级别中的奇数标签
nth-child(even)选中同级别中的偶数标签
nth-child(xn+y)选中同级别中的xn+y个标签，n从0开始自增，如：2n+1，3n+0等函数

格式2：p:first-of-type{color: red;}
作用2：选中级别中同类型的第n个标签
last-of-type,nth-of-type(n),nth-last-of-type(n)，nth-of-type(odd)，nth-of-type(even)

九、属性选择器

1.根据指定的属性名称找到对应的标签，然后设置属性，如：
p[class]{font-size: 30px;}给有class属性的标签设置字体大小
p[class=”pa2”]
input[type=”password”]指定<input type="password" name="" id="">
2.属性的取值以什么开头
[attribute|=value] css2中的用法
[attribute^=value] css3中的用法
3.属性的取值以什么结尾
[attribute$=value] css3中的用法
4.属性的取值是否包含某个特定的值
[attribute~=value] css2中的用法
[attribute*=value] css3中的用法
如：p[class^=pa]{……}

十、通配符选择器

1.格式：{……}给当前页面所有标签设置属性，如： {margin: 0;padding:0;}
2.若当前页面标签较多，通配符选择器性能变差，企业开发中一般不使用

十一、结构性伪类选择器

1.root
:root{background:orange} 等同于html {background:orange;}
2.:not否定选择器，可以选择除某个元素之外的所有元素
如给表单中除submit按钮之外的input元素添加红色边框，CSS代码可以写成：

input:not([type=”submit”]){border:1px solid red;}
