压缩命令

发表于 2017-02-08   |   分类于 基础   |  

linux基础知识，常用压缩打包及解压缩解打包命令

Linux常见的压缩格式有.zip .gz .bz2 .tar.gz .ter.bz2

.zip格式压缩

压缩文件

zip 压缩文件名 源文件

压缩目录

zip -r 压缩文件名 源目录

解压缩

unzip 压缩文件

.gz格式压缩

压缩文件，源文件会消失

gzip 源文件

压缩文件，源文件保留

gzip -c 源文件 > 压缩文件

压缩目录下的子文件，gzip不能压缩文件

gzip -r 目录

解压缩文件

gzip -d 压缩文件

.bz2格式压缩

压缩文件，源文件会消失

bzip2 源文件

压缩文件，源文件保留

bzip2 -k 源文件

bzip2不能压缩目录，会出错

tar打包

打包是为了解决目录无法通过gzip,bzip2压缩的问题

tar -zcvf 压缩包名.tar.gz 源文件 #压缩.gz文件
tar -zxvf 压缩包名.tar.gz #解压缩.gz文件
tar -jcvf 压缩包名.tar.bz2 源文件 #压缩.bz2文件
tar -jxvf 压缩包名.tar.bz2 #解压缩.bz2文件

如果想指定解压缩位置，在后面加-C目标位置即可，如tar -jxvf jp.tar.bz2 -C /tem/
