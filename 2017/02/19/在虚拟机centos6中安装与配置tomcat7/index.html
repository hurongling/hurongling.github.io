在虚拟机centos6中安装与配置tomcat7

发表于 2017-02-19   |   分类于 操作   |  

环境简述：通过xshell连接虚拟机，在虚拟机centos6中安装与配置tomcat7
参考

tomcat官网
根据需要选择tomcat版本，我选择的是tomcat7，下载.tar.gz包，我的做法与上篇配置安装jdk类似，先下载到windows本地，再通过xshell的rz命令上传到/usr/local/目录下
接着，利用#tar –zxvf apache-tomcat-7.0.61.tar.gz解压缩
为了方便，我用mv重命名 #mv apache-tomcat-7.0.61 tomcat
因为前面已经配置了环境变量，现在可以cd到tomcat目录下的bin目录，输入

./startup.sh 启动tomcat
这时候，在windows下的浏览器输入虚拟机IP及端口号8080访问tomcat，发现访问不了

真相只有一个！凶手就是。。。强行柯南上身。。

打开windows下的cmd，ping一下虚拟机的IP,发现ping得通
试着telnet一下虚拟机的ip和8080端口，发现无法连接，哈哈，如果我的推理没有错的话，问题出在端口这里！
在虚拟机里输入netstat –ntlp，发现8080端口有在监听，那么可以判断，问题不是出在tomcat的配置上，那么很可能问题出在防火墙上！
修改centos6系统防火墙配置vim /etc/sysconfig/iptables，使得防火墙允许8080端口的访问
加入 -A RH-Firewall-1-INPUT -m state --state NEW -m tcp -p tcp --dport 8080 -j ACCEPT
就可以了，然后重新启动linux的防火墙服务
service iptables restart
再试着在windows的浏览器下输入虚拟机的ip加8080端口，访问成功！
