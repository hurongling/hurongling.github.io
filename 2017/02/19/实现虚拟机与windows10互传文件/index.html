实现虚拟机与windows10互传文件

发表于 2017-02-19   |   分类于 操作   |  

我的Windows10安装了putty，想传文件到我的虚拟机，知道有xshell的存在，嫌麻烦，直接通过putty连接虚拟机，安装sz，rz后发现传不了文件。原来，PuTTY由于其局限性，不支持rz和sz命令，所以需要下载改进后的Le PuTTY来实现这个功能。或者，使用xshell。看了网上其他人对xshell的一致好评，我决定，果断放弃putty，下载xshell

xshell官网下载地址：http://www.netsarang.com/download/down_xsh.html
安装的时候，请选择安装免费版“Free For Home/School”，因为Xshell在商业环境下使用是收费的
安装完后的连接操作与putty大同小异。连接到虚拟机后，在虚拟机输入rpm -qa |grep sz 和 rpm -qa|grep rz 命令查看是否已安装该程序

[root@localhost git_project]# rpm -qa|grep sz
lrzsz-0.12.20-27.1.el6.x86_64
[root@localhost git_project]# rpm -qa|grep rz
lrzsz-0.12.20-27.1.el6.x86_64

如上则表示已安装，没有安装回车不会有所显示
输入yum -y install lrzsz* 进行安装，如果显示

No package rz available.
Error: Nothing to do

不怕，再输入一次yum -y install lrzsz*就可以成功安装了

  • 输入sz filename把虚拟机的文件传到本地电脑，回车即有文件选择框让你选择文件存放位置
  • 输入rz 把电脑文件传到虚拟机，回车即有文件选择款让你选择上传文件
    rz和sz传文件的速度较慢，但对于较小的文件已够用，如果文件的传输速度是0kb/sce，可能是卡住了，取消再重传就行了，这事我有发言权，毕竟遇到过几次[捂脸哭.gif]
