Canvas基础知识

发表于 2017-02-25   |   分类于 css3   |  

HTML5 中 canvas 标签用于绘制图像（通常是通过 JavaScript 脚本进行绘制）。canvas 元素本身并没有绘制能力（它仅仅是图形的容器），必须使用脚本来完成实际的绘图任务。
getContext() 方法可返回一个对象，该对象提供了用于在画布上绘图的方法和属性。getContext(“2d”) 对象属性和方法，可以在画布上绘制文本、线条、矩形、圆形等等。
canvas 提供了很多强大的属性和方法，实际项目开发中，常用于 HTML5 游戏开发、图形、图像开发等。Internet Explorer 9+、Firefox、Opera、Chrome 以及 Safari 支持 canvas 及其属性和方法。

##一、颜色、样式和阴影

  1. fillStyle 属性设置或返回用于填充绘画的颜色、渐变或模式。
    context.fillStyle=color|gradient|pattern;
  2. strokeStyle 属性设置或返回用于笔触的颜色、渐变或模式。
    context.strokeStyle=color|gradient|pattern;
  3. context.shadowBlur=number; 属性设置或返回阴影的模糊级数。
  4. context.shadowColor=color;属性设置或返回用于阴影的颜色。
  5. context.shadowOffsetX=number; 设置或返回形状与阴影的水平距离。
  6. context.shadowOffsetY=number; 设置或返回形状与阴影的垂直距离。
    请将 shadowColor 属性与 shadowBlur 属性一起使用，来创建阴影。
    请通过使用 shadowOffsetX 和 shadowOffsetY 属性来调节阴影效果。
    例如： 
    var c=document.getElementById("myCanvas");
var ctx=c.getContext("2d");
ctx.shadowBlur=10;
ctx.shadowOffsetX=20;
ctx.shadowColor="black";
ctx.fillStyle="blue";
ctx.fillRect(20,20,100,80);
  7. context.createLinearGradient(x0,y0,x1,y1); 创建线性的渐变对象
    提示：请使用该对象作为 strokeStyle 或 fillStyle 属性的值
    提示：请使用 addColorStop() 方法规定不同的颜色，以及在 gradient 对象中的何处定位颜色
    示例：定义从黑到白的渐变（从左向右），作为矩形的填充样式
    ```javascript
    // html

// javascript
var c=document.getElementById(“myCanvas”);
var ctx=c.getContext(“2d”);
var grd=ctx.createLinearGradient(0,0,170,0);
grd.addColorStop(0,”black”);
grd.addColorStop(1,”white”);
ctx.fillStyle=grd;
ctx.fillRect(20,20,150,100);
```

  1. context.createPattern(image,”repeat|repeat-x|repeat-y|no-repeat”);在指定的方向内重复指定的元素。元素可以是图片、视频，或者其他 <canvas> 元素。被重复的元素可用于绘制/填充矩形、圆形或线条等等
  2. context.createRadialGradient(x0,y0,r0,x1,y1,r1); 创建放射状/圆形渐变对象。
    参数分别为：渐变的开始圆的 x 坐标，y 坐标，开始圆的半径以及结束圆的 x 坐标，y 坐标，结束圆的半径
  3. gradient.addColorStop(stop,color);
    Stop:介于 0.0 与 1.0 之间的值，表示渐变中开始与结束之间的位置
    ##二、线条样式
  4. context.lineCap=”butt|round|square”;设置或返回线条末端线帽的样式
  5. context.lineJoin=”bevel|round|miter”;设置或返回所创建边角的类型，当两条线交汇时
  6. context.lineWidth=number; 设置或返回当前线条的宽度，以像素计
  7. context.miterLimit=number; 提示：只有当 lineJoin 属性为 “miter” 时，miterLimit 才有效
    ##三、矩形
  8. context.rect(x,y,width,height); 创建矩形。提示：请使用 stroke() 或 fill() 方法在画布上实际地绘制矩形。
  9. context.fillRect(x,y,width,height);绘制“已填色”的矩形。默认的填充颜色是黑色。提示：请使用 fillStyle 属性来设置用于填充绘图的颜色、渐变或模式。
  10. context.strokeRect(x,y,width,height); 绘制矩形（不填色）。笔触的默认颜色是黑色。
    提示：请使用 strokeStyle 属性来设置笔触的颜色、渐变或模式
  11. context.clearRect(x,y,width,height); 清空给定矩形内的指定像素。
    ##四、路径
  12. context.fill();填充当前的图像（路径）。默认颜色是黑色。提示：请使用 fillStyle 属性来填充另一种颜色/渐变。
  13. context.stroke();绘制路径。默认颜色是黑色。提示：请使用 strokeStyle 属性来绘制另一种颜色/渐变。
  14. context. beginPath(); 开始一条路径，或重置当前的路径。
  15. context.moveTo(x,y);
  16. context.closePath();创建从当前点到开始点的路径
  17. context.lineTo(x,y);该方法并不会创建线条
  18. context.clip();从原始画布中剪切任意形状和尺寸
  19. context.quadraticCurveTo(cpx,cpy,x,y); 贝塞尔控制点的x,y坐标,结束点的x,y坐标
    请使用 beginPath() 和 moveTo(x,y) 方法来定义开始点。
  20. context.bezierCurveTo(cp1x,cp1y,cp2x,cp2y,x,y);
  21. context.arc(x,y,r,sAngle,eAngle,counterclockwise); 创建弧/曲线
    提示：请使用 stroke() 或 fill() 方法在画布上绘制实际的弧
    圆的中心的x,y坐标,半径，起始角，以弧度计。（三点钟位置是0度，无论如何都是顺时针增大）。结束角，counterclockwise（可选，规定应该逆时针还是顺时针绘图。False = 顺时针，true = 逆时针，默认是顺时针）
  22. arcTo(x1,y1,x2,y2,r) 方法在画布上创建介于两个切线之间的弧/曲线。
  23. context.isPointInPath(x,y); 如果指定的点位于当前路径中方法返回true;否则返回 false
    ##五、方法
  24. context.scale(scalewidth,scaleheight);
  25. context.rotate(angle); 旋转当前的绘图。
  26. context.translate(x,y); 重新映射画布上的 (0,0) 位置
  27. context.transform(a,b,c,d,e,f);
  • a 水平缩放绘图
  • b 水平倾斜绘图
  • c 垂直倾斜绘图
  • d 垂直缩放绘图
  • e 水平移动绘图
  • f 垂直移动绘图
    前面由 rotate(), scale(), translate(), or transform()带来的坐标改变会对他造成影响，会加倍作用。
  1. context.setTransform(a,b,c,d,e,f);同transform(a,b,c,d,e,f);但不会受前面坐标变化的影响
    ##六、文本
  2. context.font=”italic small-caps bold 12px arial”;设置字体属性，有多个
  3. context.textAlign=”center|end|left|right|start”;设置或返回文本内容的当前对齐方式
  4. context.textBaseline=”alphabetic|top|hanging|middle|ideographic|bottom”;设置或返回在绘制文本时的当前文本基线
  5. context.fillText(text,x,y,maxWidth);在画布上绘制填色的文本，maxWidth可选
  6. context.strokeText(text,x,y,maxWidth); 在画布上绘制没有填色的文本
  7. context.measureText(text).width; 返回包含指定文本宽度的对象
    ##七、图像绘制
  8. 在画布上定位图像：context.drawImage(img,x,y);
  9. 在画布上定位图像，并规定图像的宽度和高度context.drawImage(img,x,y,width,height)
  10. 剪切图像，并在画布上定位被剪切的部分：context.drawImage(img,sx,sy,swidth,sheight,x,y,width,height);
