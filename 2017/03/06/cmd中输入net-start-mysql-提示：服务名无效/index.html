cmd中输入net start mysql 提示：服务名无效，可能原因及解决办法

可能原因1：

你的mysql服务名不叫mysql
解决方法：右击我的电脑—>管理—>服务和应用程序—>服务—>服务
看下你的mysql服务名称是什么，我的是MySQL57（服务名不区分大小写）
输入net start mysql57
就可以了~
如果显示“发生系统错误 5”那是因为当前用户的操作权限太低了，切换到管理员模式就可以启动了。在开始菜单的搜索框输入cmd，然后右键单击，并选择以管理员身份运行，再输入net start mysql57 就可以了。

可能原因2：

mysql服务还没安装成功
解决方法：
用cmd请进入MySQL的bin目录，并在bin目录打开命令行窗口，在命令行窗口输入：mysqld –install，回车，提示：Service successfully installed。表示安装MySQL服务成功，命令行窗口输入：net start mysql ，可以正常启动。

再写一点点，如果在cmd中登陆mysql的时候出现“Mysql不是内部或外部命令，也不是可运行的程序或其处理文件” 而在mysql安装目录下的bin目录那里就能操作mysql，那原因就是你还没配置环境变量，把mysql安装目录下的bin目录路径保存到PATH环境变量中就可以啦！
