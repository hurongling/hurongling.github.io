修改mysql数据表ALTER的用法

添加单列：
ALTER TABLE users1(表名称) ADD password(要添加的列的名称) VARCHAR(32) NOT NULL AFTER username(添加到username列的后面); //AFTER指定添加位置，添加到某数据库的某个数据表的某个列后
添加多列：
ALTER TABLE tbl_name ADD (col_name column_definition,...); //添加多列不能指插入定位置，默认放在最后
删除单列：
ALTER TABLE tbl_name(表名称) DROP col_name(列名称);
删除多列：
ALTER TABLE users1(表名称) DROP password(列名称),DROP age(列名称);
添加主键约束：
ALTER TABLE users2(表名称) ADD [CONSTRAINT [symbol]] PRIMARY KEY (id(列名称)); //symbol为PRIMARY KEY的别名
删除主键约束：
ALTER TABLE users2 DROP PRIMARY KEY;
添加唯一约束：
ALTER TABLE users2(表名称) ADD UNIQUE (username(列名称));
删除唯一约束：
ALTER TABLE users2(表名称) DROP INDEX username;//用命令SHOW INDEXES FROM users2(表名称)\G;查看唯一约束的列名称
添加外键约束：
ALTER TABLE users2(表名称) ADD FOREIGN KEY (pid(表users2中的列的列名称)) REFERENCES provinces(表名称) (id(表provinces中的列的列名称));
删除外键约束：
ALTER TABLE users2 DROP FOREIGN KEY users2_ibfk_1;//用SHOW CREATE TABLE users2;查看外键约束
添加默认约束：
ALTER TABLE users2(表名称) ALTER age(列名称) SET DEFAULT 15;//把users2中age的默认值改为15
删除默认约束：
ALTER TABLE users2(表名称) ALTER age(列名称) DROP DEFAULT;//把users2中age的默认值删除清空，变为NULL
修改列定义：
ALTER TABLE users2 MODIFY id SMALLINT UNSIGNED NOT NULL FIRST;//修改id字段的数据类型和在表users2中所处的位置
修改列名称：
ALTER TABLE users2 CHANGE pid(旧的字段名称) p_id(新的字段名称) TINYINT UNSIGNED NOT NULL;//修改pid字段，改为p_id字段
数据表更名：
ALTER TABLE users2 RENAME users3;//把表名users2改为新的users3
