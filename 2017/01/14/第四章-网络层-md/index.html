计算机网络（第6版）第四章 网络层学习笔记

一.网络层提供的两种服务

“面向连接”虚电路（Virtual Circuit）服务和“无连接”的数据报服务

二.网际协议IP

1.与IP协议配套使用的三个协议

  • 地址解析协议ARP
  • 网际控制报文协议ICMP
  • 网际组管理协议IGMP
    逆地址解析协议已被淘汰
    2.虚拟互联网络
    网络互联的一些中间设备
  • 转发器，集线器——物理层
  • 网桥，桥接器——数据链路层
  • 路由器——网络层
  • 网关——网络层以上
    互联网可以由多种异构网络互联组成
    3.分类的IP地址
    IP地址::={<网络号>,<主机号>}
  • A类地址：第一个字节表示的范围1-126
  • B类地址：128-191（127开头的网络号不是一个网络地址，保留为本地软件环回测试）
  • C类地址：192-233
  • D类地址：224-239，为多播地址
  • E类地址：保留为今后用
    近年使用无分类IP地址，A，B，C类已成为历史
    4.一般不使用的特殊IP地址
网络号 主机号 原地址使用 目的地址使用 代表意思
0 0 可以 不可 本网络上的本主机
0 hostid 可以 不可 本网络上某个主机hostid
全1 全1 不可 可以 只在本网络进行广播
netid 全1 不可 可以 对netid上的所有主机进行广播
127 非全0或全1的任何数 可以 可以 用作本地软件环回测试只用
